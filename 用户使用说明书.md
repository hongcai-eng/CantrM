## 1. 系统概述

本系统是一套多租户合同管理平台，支持合同全生命周期管理，包括合同签订、收付款跟踪、交付记录、发票管理及统计分析。

**合同生命周期：**

```
新建合同 → 收付款记录 → 交付记录 → 开票记录 → 自动完结
```

当已收付款总额 ≥ 合同总价，且已开具发票总额 ≥ 合同总价时，合同状态自动变为「已完结」。

## 2. 登录与退出

### 登录

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击「登录」

> 登录页会自动显示所属公司的名称和 Logo。

### 退出

点击页面右上角的用户名，选择「退出登录」。

## 3. 合同管理

### 3.1 合同列表

进入系统后默认显示合同列表，支持以下筛选条件：

| 筛选项 | 说明 |
|--------|------|
| 项目负责人 | 模糊匹配 |
| 客户名称 | 模糊匹配 |
| 合同类型 | 下拉选择 |
| 履约状态 | 进行中 / 已完结 |
| 业务类型 | 销售 / 采购 |
| 签订年份 | 下拉选择 |
| 发票状态 | 已开具 / 未开具 |
| 预警 | 超过 30 天未收付款的进行中合同 |

筛选后页面底部显示汇总统计：合同数量、合同总价、已/未收付款、已/未开票金额。

### 3.2 新建合同

1. 点击「新建合同」按钮
2. 填写基本信息：

   | 字段 | 说明 |
   |------|------|
   | 合同编号 | 可选，作为合同唯一标识 |
   | 客户名称 | 输入关键字自动搜索，未找到可跳转客户管理新增 |
   | 项目名称 | 必填 |
   | 合同总价 | 必填，数值 |
   | 合同类型 | 下拉选择 |
   | 业务类型 | 销售 / 采购 |
   | 项目负责人 | 必填 |
   | 销售人员 | 可选 |
   | 签订日期 | 日期选择 |

3. 上传合同文件（可选，支持任意格式）
4. 填写产品明细（可添加多行）：

   | 字段 | 说明 |
   |------|------|
   | 产品名称 | 输入关键字自动搜索，未找到可跳转产品管理新增 |
   | 产品类型 | 硬件设备 / 软件 / 技术服务 / 技术开发 |
   | 型号 | 可选 |
   | 单位 | 可选 |
   | 数量 | 数值 |
   | 单价 | 数值 |
   | 小计 | 自动计算 |
   | 发票税率 | 如 13% |

5. 点击「保存」

### 3.3 查看合同

在合同列表点击「查看」，进入合同详情页，可查看：
- 基本信息
- 产品明细
- 收付款记录
- 交付记录
- 发票记录

### 3.4 编辑合同

在合同列表点击「编辑」，修改合同信息后点击「保存」。

### 3.5 删除合同

在合同详情页点击「删除合同」，确认后删除（不可恢复）。

## 4. 收付款管理

在合同详情页「收付款记录」区域操作。

### 新增收付款记录

1. 点击「新增收付款」
2. 填写：金额、日期、类型（收款/付款）、备注
3. 可上传回单凭证文件
4. 点击「保存」

### 回单文件操作

每条记录的回单文件支持：**预览** | **下载** | **删除**

### 删除记录

点击记录行的「删除记录」按钮，删除整条记录（含回单文件）。删除后系统自动重新计算未付款金额和合同状态。

---

## 5. 交付管理

在合同详情页「交付记录」区域操作。

### 新增交付记录

1. 点击「新增交付」
2. 填写：交付日期、交付内容、备注
3. 可上传交付文件
4. 点击「保存」

### 交付文件操作

每条记录的文件支持：**预览** | **下载** | **删除**

### 删除记录

点击「删除记录」按钮，删除整条记录（含交付文件）。

---

## 6. 发票管理

在合同详情页「发票记录」区域操作。

### 新增发票记录

1. 点击「新增发票」
2. 填写：

   | 字段 | 说明 |
   |------|------|
   | 发票金额 | 数值 |
   | 开票日期 | 日期选择 |
   | 发票号 | 系统自动检测重复 |
   | 发票类型 | 普票 / 专票 |
   | 发票状态 | 已开具 / 未开具 |

3. 可上传发票文件
4. 点击「保存」

### 发票文件操作

每条记录的文件支持：**预览** | **下载** | **删除**

### 删除记录

点击「删除记录」按钮，删除整条记录（含发票文件）。删除后系统自动重新计算未开票金额和合同状态。

## 7. 统计分析

点击左侧菜单「统计分析」进入。

### 筛选条件

支持按项目负责人、客户名称、合同类型、业务类型、履约状态、签订年份筛选。

### 统计维度

可勾选显示以下统计表格：
- 按项目负责人统计
- 按客户统计
- 按合同类型统计
- 按业务类型统计
- 按履约状态统计
- 合同明细数据

### 导出 Excel

点击「导出 Excel」，支持三种布局：

| 方案 | 说明 |
|------|------|
| 方案 A | 单 Sheet，各维度纵向排列 |
| 方案 B | 单 Sheet，各维度横向并排 |
| 方案 C | 多 Sheet，每个维度独立一页 |

导出内容包含 16 个字段：合同编号、客户名称、项目名称、产品名称、合同总价、发票税率、合同类型、业务类型、项目负责人、销售人员、签订日期、状态、已收付款、未收付款、已开票、未开票。

> 需要「导出EXCEL」权限。

## 8. 客户管理

点击左侧菜单「客户管理」进入。

### 主要操作

- **搜索**：按客户名称或省份筛选
- **新增客户**：填写名称、省份、地区、统一信用代码
- **编辑**：修改客户信息
- **删除**：删除客户记录

> 新建合同时输入客户名称若未找到，可直接跳转此页面新增，完成后自动返回合同表单并选中。

## 9. 产品管理

点击左侧菜单「产品管理」进入。

### 主要操作

- **搜索**：按产品名称、分类、型号筛选
- **新增产品**：填写名称、分类、型号、单位、税率、参考数量、参考单价
- **编辑**：修改产品信息
- **删除**：删除产品记录

> 新建合同时输入产品名称若未找到，可直接跳转此页面新增，完成后自动返回合同表单并选中。

## 10. 导入 Excel

> 需要「导入EXCEL」权限。

点击合同列表页「导入 Excel」按钮进入。

### 操作步骤

1. 下载导入模板（页面提供示例格式说明）
2. 按格式填写 Excel 文件
3. 点击「选择文件」上传
4. 系统自动检测重复合同并提示
5. 确认后点击「导入」

### Excel 格式说明

**必填列：**

| 列名 | 说明 |
|------|------|
| 客户名称 | 必填 |
| 项目名称 | 必填 |
| 合同总价 | 必填，数值 |

**可选列：**

| 列名 | 说明 |
|------|------|
| 合同编号 | 用于重复检测的唯一标识 |
| 签订日期 | 支持多种日期格式 |
| 合同类型 | |
| 业务类型 | 销售 / 采购，默认「销售」 |
| 项目负责人 | 也可写「项目人员」 |
| 销售人员 | |
| 状态 | 进行中 / 已完结，默认「进行中」 |
| 产品名称 | |
| 型号 | |
| 单位 | |
| 数量 | 数值 |
| 单价 | 数值 |
| 发票税率 | 如 0.13 表示 13% |

**多产品合同：** 同一合同的多个产品占多行，合同级字段在第一行填写，后续行留空即可（系统自动向下填充）。

## 11. 组织管理

> 仅租户管理员可操作。

点击左侧菜单「组织管理」进入。

### 主要操作

- **新建组织**：填写名称、描述、上级组织，并勾选该组织的权限项
- **编辑组织**：修改组织信息和权限
- **删除组织**：有子组织或成员时不可删除
- **人员调动**：将用户调入或调出某组织，调入时用户自动继承该组织的权限

### 权限项说明

新建/编辑组织时可勾选以下权限，组织成员将自动获得对应权限：

增加、删除、修改、查阅、上传、下载、导入EXCEL、导出EXCEL

---

## 12. 用户管理

> 仅租户管理员可操作。

点击左侧菜单「用户管理」进入。

### 主要操作

- **新增用户**：填写用户名、密码、角色，勾选权限项
- **编辑用户**：修改用户名、角色、权限，或重置密码
- **删除用户**：内置管理员账号不可删除

## 13. 系统配置

> 仅租户管理员可操作。

点击左侧菜单「系统配置」进入，可设置：
- 公司名称（显示在登录页和系统顶部）
- 公司 Logo（上传图片文件）

---

## 14. 租户管理

> 仅总超级管理员（superadmin）可操作。

登录 superadmin 账号后进入租户管理界面。

### 主要操作

- **新建租户**：填写租户名称、描述，同时创建该租户的管理员账号
- **编辑租户**：修改租户名称、描述、管理员账号信息
- **删除租户**：同时删除该租户下所有用户（不可恢复）
- **品牌设置**：为租户设置公司名称和 Logo
- **查看用户**：查看某租户下的用户列表，可重置用户密码

---

## 15. 权限说明

### 角色类型

| 角色 | 说明 |
|------|------|
| 总超级管理员 | 系统唯一，管理所有租户，不访问合同数据 |
| 超级管理员 | 每个租户的最高管理员，拥有全部权限 |
| 普通用户 | 按分配的权限项操作 |

### 权限项说明

| 权限项 | 控制范围 |
|--------|---------|
| 增加 | 新建合同、客户、产品、收付款、交付、发票记录 |
| 删除 | 删除合同、客户、产品 |
| 修改 | 编辑合同、客户、产品，进入合同管理页 |
| 查阅 | 预览上传的附件文件 |
| 下载 | 下载附件文件 |
| 上传 | 上传附件文件 |
| 导入EXCEL | 使用 Excel 批量导入合同 |
| 导出EXCEL | 导出合同列表或统计分析为 Excel |

> 超级管理员自动拥有全部权限，无需单独配置。
> 
> 无权限时操作会提示：**「你没有此项权限，请与管理员联系」**
