# 合同管理系统功能更新说明

## 📋 本次更新内容

### 1. 多产品合同支持 ✅
**新建/编辑合同时，支持添加多个产品，每个产品可独立设置合同类型**

#### 新增功能：
- ✅ 动态添加/删除产品条目
- ✅ 每个产品可选择：工程/货物/服务
- ✅ 自动计算产品小计和合同总价
- ✅ 支持产品名称、型号、单位、数量、单价、税率

#### 数据库变更：
- **新增表**：`contract_product` - 存储合同的多个产品信息
- **新增字段**：
  - `contract_product.contract_type` - 每个产品的合同类型
  - `contract_product.subtotal` - 产品小计

---

### 2. 多租户架构 ✅
**Superadmin可创建多个独立的客户租户，每个租户数据完全隔离**

#### 三级权限体系：
1. **Superadmin（总超级管理员）**
   - 账号：`superadmin` / 密码：`654321`
   - 可创建/管理所有租户客户
   - 可查看所有数据
   - 独有"租户管理"菜单

2. **客户超级管理员（Admin）**
   - 由Superadmin创建，绑定到特定租户
   - 只能管理本租户的用户、客户、产品、合同
   - 不能访问其他租户数据
   - 可创建本租户的普通用户

3. **普通用户**
   - 由客户超级管理员创建
   - 继承所属租户的数据权限
   - 根据角色权限操作数据

#### 数据库变更：
- **新增表**：`tenant_customer` - 租户客户表
- **新增字段**（所有表）：
  - `user.customer_id` - 用户所属租户
  - `customer.customer_id` - 客户信息所属租户
  - `product.customer_id` - 产品所属租户
  - `contract.customer_id` - 合同所属租户

---

### 3. 导入功能优化 ✅
**Excel导入支持多产品，重复检测更严格**

#### 改进点：
- ✅ 支持工程/货物/服务组合导入
- ✅ 检测到重复合同时，只显示提示，不允许确认导入（去掉"确认重复导入"按钮）
- ✅ 自动创建 `contract_product` 记录
- ✅ 数据隔离：只在当前租户内检测重复

---

### 4. 导出功能增强 ✅
**合同列表和统计分析导出时，增加下载完成提示**

#### 改进点：
- ✅ 合同列表导出：显示"正在生成" → "下载完成"提示
- ✅ 统计分析导出：勾选导出项 → 显示"正在生成" → "下载完成"提示
- ✅ 统计分析支持按筛选条件导出（项目负责人、客户、合同类型、业务类型、履约状态、签订年份）

---

### 5. 界面优化 ✅
**删除"具体子类"字段，保持界面简洁**

#### 改进点：
- ✅ 新建/编辑合同表单中移除"具体子类项"输入框
- ✅ 保留原有字段结构（数据库中仍保留，只是界面不显示）

## 🚀 使用说明

### 1. 数据库迁移（必须执行）
```bash
python migrate_db.py
```

### 2. 启动应用
```bash
python app.py
```

### 3. 默认账号
- **总超级管理员**：`superadmin` / `654321`
- **普通超级管理员**：`admin` / `123456`

### 4. 创建租户流程
1. 使用 `superadmin` 登录
2. 进入"租户管理"
3. 点击"新建租户客户"
4. 填写租户信息和管理员账号
5. 创建成功后，使用该管理员账号登录即可管理该租户数据

## ⚠️ 重要说明

### 数据兼容性
- ✅ 原有合同数据保留，`customer_id` 为 NULL 表示未分配租户
- ✅ Superadmin 和原 admin 账号可查看所有数据
- ✅ 新创建的租户数据完全隔离

### 多产品功能
- ✅ 新建合同时至少添加一个产品
- ✅ 编辑合同时会重新保存所有产品（删除旧的，添加新的）
- ✅ 合同总价需手动输入或点击"自动计算"按钮

### 权限说明
- ✅ 只有 Superadmin 可以创建租户
- ✅ 客户超级管理员不能创建其他超级管理员
- ✅ 普通用户权限由创建者设置
