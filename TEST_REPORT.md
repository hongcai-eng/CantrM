## 使用说明

### 1. 设置租户品牌
1. 使用 `superadmin` 登录
2. 进入"租户管理"
3. 点击"设置品牌"按钮
4. 上传 Logo 并填写公司名称

### 2. 管理组织结构
1. 使用客户超级管理员登录（如 admin1）
2. 点击"组织管理"菜单
3. 点击"+ 新建组织"创建组织
4. 可选择上级组织创建层级结构
5. 使用"人员调动"功能分配用户到组织

### 3. 使用筛选统计
1. 进入"合同列表"
2. 使用筛选区域选择条件
3. 点击"筛选"查看结果
4. 统计汇总卡片自动更新
5. 点击"导出Excel"导出当前筛选结果
