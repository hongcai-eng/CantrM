# 合同管理系统 - 启动指南

## 🚀 快速启动

### 1. 首次启动（必须执行数据库迁移）

```bash
# 步骤1：执行数据库迁移
python migrate_db.py

# 步骤2：启动应用
python app.py
```

### 2. 后续启动

```bash
python app.py
```

访问地址：http://127.0.0.1:5000

## 📋 功能使用流程

### 创建租户客户
1. 使用 `superadmin` 登录
2. 点击导航栏"租户管理"
3. 点击"新建租户客户"
4. 填写：
   - 租户名称（如：XX公司）
   - 描述（可选）
   - 客户超级管理员账号
   - 初始密码
5. 创建成功后，该管理员可登录管理本租户数据

### 新建多产品合同
1. 点击"新建合同"
2. 填写客户名称、项目名称
3. 在"产品信息"区域：
   - 填写产品名称、合同类型（工程/货物/服务）
   - 填写型号、单位、数量、单价
   - 系统自动计算小计
4. 点击"+ 添加产品"可添加更多产品
5. 点击"自动计算总价"或手动输入合同总价
6. 填写其他信息后保存

### 导入Excel合同
1. 点击"导入Excel"
2. 选择符合格式的Excel文件
3. 系统自动检测重复：
   - 如有重复，显示提示并要求重新上传
   - 无重复则直接导入
4. 每行数据创建一个合同记录（支持多产品）

### 导出数据
1. **合同列表导出**：
   - 点击"导出Excel"
   - 显示"正在生成"提示
   - 文件自动下载
   - 显示"下载完成"提示

2. **统计分析导出**：
   - 点击"导出Excel"
   - 勾选需要导出的统计项
   - 点击"确认导出"
   - 显示"正在生成" → "下载完成"

## 🔒 权限说明

### 三级权限体系

| 角色 | 权限范围 | 数据隔离 |
|------|---------|---------|
| Superadmin | 全局管理 | 无限制 |
| 客户超级管理员 | 本租户管理 | 只能访问本租户数据 |
| 普通用户 | 根据角色权限 | 继承所属租户权限 |

### 数据隔离规则
- ✅ 客户超级管理员创建的所有数据自动关联到其租户
- ✅ 客户超级管理员只能查看/编辑本租户数据
- ✅ 不同租户之间数据完全隔离
- ✅ Superadmin 和原 admin 可查看所有数据

## ⚠️ 注意事项

1. **首次启动必须执行迁移**
   - 如果直接运行 `python app.py` 会报错
   - 必须先运行 `python migrate_db.py`

2. **多产品合同**
   - 至少添加一个产品
   - 合同总价需手动输入或点击"自动计算"

3. **导入Excel格式**
   - 必须包含：客户名称、项目名称、合同总价
   - 产品信息可选
   - 每行创建一个独立合同记录

4. **租户管理**
   - 只有 Superadmin 可以创建租户
   - 客户超级管理员不能创建其他超级管理员
   - 租户创建后不能删除（需手动操作数据库）

## 🐛 常见问题

### Q: 启动时报错 "no such column: user.customer_id"
**A**: 未执行数据库迁移，运行 `python migrate_db.py`

### Q: 客户超级管理员看不到"租户管理"菜单
**A**: 正常现象，只有 Superadmin 可以访问租户管理

### Q: 导入Excel时提示重复，但想强制导入
**A**: 本次更新已移除"确认重复导入"功能，需要修改Excel数据后重新上传

### Q: 编辑合同时产品信息丢失
**A**: 编辑时会重新保存所有产品，确保填写完整后再保存
