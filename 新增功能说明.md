# 合同管理系统 - 新增功能说明

## 一、新增数据库字段

### 1. Contract（合同表）
- `business_type` - 业务类型（销售/采购）

### 2. Invoice（发票表）
- `invoice_status` - 发票开具状态（未开具/已开具/已作废）

### 3. Delivery（交付记录表）
- `delivery_file` - 交付记录文件路径

### 4. 新增表：User（用户表）
- `id` - 主键
- `username` - 用户名
- `password_hash` - 密码哈希
- `role` - 角色（超级管理员/管理人员/合同负责人/合同执行人员/合同审核监督人员）
- `permissions` - 权限（增加/删除/修改/查阅/上传/下载）
- `created_at` - 创建时间

### 5. 新增表：Customer（客户表）
- `id` - 主键
- `name` - 客户名称
- `province` - 省份
- `region` - 区域
- `created_at` - 创建时间

### 6. 新增表：Product（产品表）
- `id` - 主键
- `name` - 产品名称
- `category` - 产品类型（硬件设备/软件/技术服务/技术开发）
- `model` - 型号
- `unit` - 单位
- `created_at` - 创建时间

## 二、新增文件

### 1. 模板文件
- `templates/login.html` - 登录页面
- `templates/users.html` - 用户管理列表页
- `templates/user_form.html` - 用户添加/编辑表单
- `templates/customers.html` - 客户管理页面
- `templates/products.html` - 产品管理页面
- `templates/statistics.html` - 统计分析页面

### 2. 更新的模板文件
- `templates/base.html` - 添加导航菜单和用户信息显示
- `templates/contract_form.html` - 添加业务类型选择和自动搜索功能
- `templates/contract_detail.html` - 添加业务类型显示、发票状态、文件预览功能

## 三、新增功能逻辑

### 1. 用户登录和权限管理
- 登录/登出功能（`/login`, `/logout`）
- 超级管理员账号：admin / 123456
- 用户管理功能（`/users`, `/user/new`, `/user/<id>/edit`, `/user/<id>/delete`）
- 权限验证装饰器（`@login_required`, `@permission_required`）
- 支持6种权限：增加、删除、修改、查阅、上传、下载

### 2. 客户名称维护
- 客户列表展示（`/customers`）
- 添加客户（`/customer/new`）
- 编辑客户（`/customer/<id>/edit`）
- 删除客户（`/customer/<id>/delete`）
- 按省份区域分类显示
- 客户名称自动搜索API（`/api/customers/search`）

### 3. 产品信息维护
- 产品列表展示（`/products`）
- 添加产品（`/product/new`）
- 编辑产品（`/product/<id>/edit`）
- 删除产品（`/product/<id>/delete`）
- 支持4种产品类型：硬件设备、软件、技术服务、技术开发
- 产品信息自动搜索API（`/api/products/search`）

### 4. 文件预览功能
- 合同文件预览和下载
- 回单文件预览和下载
- 交付记录文件上传、预览和下载
- 发票文件预览和下载
- 支持PDF和图片格式预览
- 新增预览路由（`/preview/<filename>`）

### 5. 发票开具状态
- 发票状态字段：未开具/已开具/已作废
- 状态颜色标识（绿色/橙色/红色）

### 6. 采购和销售区分
- 合同业务类型字段：销售/采购
- 在合同表单和详情页显示

### 7. 统计分析功能
- 按项目人员统计（合同数量、合同总额）
- 按客户名称统计（合同数量、合同总额）
- 按合同类型统计（合同数量、合同总额）
- 按业务类型统计（合同数量、合同总额）
- 按履约状态统计（合同数量）
- 统计页面路由（`/statistics`）

### 8. 自动搜索填充
- 客户名称输入时自动搜索已有客户
- 产品名称输入时自动搜索已有产品
- 选择产品后自动填充型号和单位
- 使用JavaScript实现实时搜索

## 四、权限控制

所有合同相关操作都需要登录后才能访问，并根据用户权限控制：
- 新建合同：需要"增加"权限
- 编辑合同：需要"修改"权限
- 查看合同：需要"查阅"权限
- 下载文件：需要"下载"权限
- 预览文件：需要"查阅"权限
- 超级管理员拥有所有权限

## 五、初始化说明

系统首次运行时会自动创建：
- 超级管理员账号：admin
- 默认密码：123456
- 超级管理员可以修改自己的密码
- 超级管理员可以添加其他用户并分配权限

## 六、使用说明

1. 首次访问系统会跳转到登录页面
2. 使用 admin / 123456 登录
3. 登录后可以在导航菜单访问各个功能模块
4. 超级管理员可以在"用户管理"中添加其他用户
5. 在"客户管理"和"产品管理"中维护基础数据
6. 创建合同时可以使用自动搜索功能快速填充客户和产品信息
7. 在"统计分析"中查看各类统计数据
