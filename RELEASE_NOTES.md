## 🎯 快速开始

### 方式 1：本地测试（最简单）

1. **双击运行**（Windows）
   ```
   双击 start.bat
   ```

2. **访问系统**
   ```
   http://localhost:5000
   ```

3. **登录测试**
   ```
   账号：superadmin
   密码：654321
   ```

### 方式 2：局域网部署

1. **Windows 服务器**
   - 将整个 `cantrm` 文件夹复制到服务器
   - 双击运行 `deploy.bat`
   - 双击运行 `start.bat`
   - 通知同事访问：`http://服务器IP:5000`

2. **Linux 服务器**
   ```bash
   # 上传文件到服务器
   scp -r cantrm/ user@server:/var/www/
   
   # 登录服务器
   ssh user@server
   
   # 部署
   cd /var/www/cantrm
   bash deploy.sh
   
   # 启动
   bash start.sh
   ```

### 方式 3：生产环境部署

详见 `DEPLOYMENT_GUIDE.md` 文档。

## 📞 技术支持

### 常见问题

**Q：无法访问系统？**
- 检查服务是否启动
- 检查防火墙是否开放 5000 端口
- 尝试访问 http://localhost:5000

**Q：登录后页面显示异常？**
- 按 Ctrl+F5 强制刷新
- 清除浏览器缓存
- 尝试使用无痕模式

**Q：导出功能不起作用？**
- 检查浏览器是否阻止下载
- 查看浏览器控制台错误信息
- 确认已按 Ctrl+F5 刷新

**Q：筛选"货物"类型没有结果？**
- 已修复，请确认使用的是 v2.0 版本
- 按 Ctrl+F5 刷新页面
